# Chrome Bookmarks

This repository contains my Chrome bookmarks, automatically synced by [Bookmarked CLI](https://github.com/vivek-dodia/bookmarked-cli).

## Sync Details
- **Synced from**: Chrome Browser
- **Format**: Formatted JSON (pretty-printed for readable diffs)
- **Frequency**: Automatic on bookmark changes (500ms debounce)
- **Tool**: [Bookmarked CLI](https://github.com/vivek-dodia/bookmarked-cli)
- **Last Updated**: Auto-updated on each sync

## About the Tool

Bookmarked CLI is a minimal background service that automatically syncs Chrome bookmarks to GitHub.

Features:
- Cross-platform (Windows, macOS, Linux)
- Real-time sync with file watching
- Version control for bookmark history
- Formatted JSON for readable diffs
